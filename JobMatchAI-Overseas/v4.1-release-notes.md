# JobMatchAI v4.1 修复说明

## 发布日期：2026年4月15日

---

## 🔧 本次修复清单

### 1. CORS 配置冲突修复
- **问题**：Cloudflare Worker 覆盖后端 CORS 头，导致双重 `Access-Control-Allow-Origin`
- **修复**：Worker 不覆盖 CORS 头，直接透传后端响应
- **涉及文件**：
  - `backend_main.py` - CORS 白名单配置
  - `cloudflare-worker.js` - 移除 CORS 头覆盖
  - `deploy-frontend.sh` - Pages Function 移除 CORS 头覆盖

### 2. URL 编码修复
- **问题**：双重 `encodeURIComponent` 导致 `c++` 变成 `c%252B%252B`
- **修复**：移除手动编码，交给 `URLSearchParams` 自动处理

### 3. API 路径修复
- **问题**：前端调用 `/api/learning/recommend`，实际后端是 `/learning/recommend`
- **修复**：统一使用 `/learning/recommend`

### 4. 学习资源 B站展示
- **问题**：Skill Gap 分析和学习资源 Tab 没有显示 B站课程
- **修复**：
  - 从 `r.free` 中提取 bilibili.com 链接 → 显示在粉色 B站区块
  - 其他免费资源 → 显示在绿色免费资源区块
  - 付费资源 → 显示在金色付费资源区块

### 5. 职位搜索功能修复
- **问题**：使用相对路径 `/jobs/search`，Cloudflare 无法代理
- **修复**：改为 `API_BASE + '/jobs/search'`
