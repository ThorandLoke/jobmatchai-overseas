## 首批转化建议（立即可做）

### "Freddo巧克力"通胀课

```
标题：为什么小时候的糖果，现在变贵了？

故事：小明妈妈说，她小时候一根冰棍只要5毛钱。
小明说，现在要5块钱！涨了10倍！

原理：
- 钱变多的速度 > 东西变多的速度 = 东西变贵（通胀）
- 政府印太多钱 → 钱不值钱
- 好的通胀：经济健康发展
- 坏的通胀：钱印太多

数字游戏：
2000年：1个Freddo = 10便士
2020年：1个Freddo = 1英镑
涨了 900%！

思考：如果你的零花钱还是那么多，你是不是"变穷了"？
```

### "贝壳当钱用"货币起源课

```
标题：世界上第一批"钱"，居然是贝壳和盐！

故事：很久以前，人们只能以物易物。
张三有稻子，李四有鱼，但交换时机对不上。
于是贝壳来了！盐来了！硬币来了！

你知道吗？
- 世界上第一批硬币是公元前700年中国商朝人发明的！
- 最早的纸币是公元1000年中国宋朝人发明的！

思考题：如果明天大家说"贝壳不值钱了"，会发生什么？
```
